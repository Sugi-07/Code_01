

# 📊 Student Record Management System

## 📌 Description
This project is a simple Student Record Management System built using Python. It allows users to add, view, search, and delete student records using file handling.

## 🧠 Objective
To practice intermediate-level Python concepts such as:
- Functions
- File Handling
- Dictionaries
- User Input Handling

## 💻 Technologies Used
- Python 3

## ⚙️ Features
- ➕ Add Student Record
- 📋 View All Records
- 🔍 Search Student by ID
- ❌ Delete Student Record

## 📂 Project Structure
student_management.py  
README.md  

## ▶️ How to Run
1. Install Python
2. Clone the repository:
   git clone https://github.com/sugi-07/code_01.git
3. Navigate to the folder:
   cd code_01
4. Run the program:
   python student_management.py
